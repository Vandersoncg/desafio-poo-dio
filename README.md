# Desafio POO – Java (DIO)

Projeto desenvolvido como parte do desafio  
**Aprendendo na Prática Programação Orientada a Objetos**, da Digital Innovation One.

## Conceitos aplicados
- Abstração
- Encapsulamento
- Herança
- Polimorfismo

## Estrutura do projeto
- `Bootcamp`
- `Curso`
- `Mentoria`
- `Dev`
- `Conteudo` (classe abstrata)

## Tecnologias
- Java 11
- Git & GitHub
- GitHub Codespaces

## Execução
O projeto é executado a partir da classe `Main.java`.
